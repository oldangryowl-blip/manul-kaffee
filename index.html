Налаштування ШІ

Щоб я міг генерувати уроки, мені потрібен твій API-ключ Google Gemini. Це безкоштовно!

Як отримати ключ:

  1. Перейди на aistudio.google.com
  2. Увійди з Google-акаунтом
  3. Натисни «Get API key» → «Create API key»
  4. Скопіюй ключ і встав сюди

⚠️ Текст запитів надсилатиметься до Google Gemini за твоїм ключем. Ключ зберігається лише на твоєму пристрої. На безкоштовному тарифі Google може використовувати запити для покращення моделей.
