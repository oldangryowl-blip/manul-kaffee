# Манул сказав: Kaffee — Встановлення на Android

## Що потрібно

- Телефон на Android (Chrome 80+)
- Комп'ютер для запуску сервера (або будь-який хостинг)
- Google Gemini API ключ (безкоштовний) — для AI-функцій, або можна використовувати демо-режим без ключа

---

## Варіант 1: Встановлення через локальну мережу

### Крок 1. Запустіть сервер на комп'ютері

```bash
cd /home/oldowl/Manul
python3 -m http.server 8082
```

Сервер запуститься на порту 8082.

### Крок 2. Дізнайтесь IP-адресу комп'ютера

```bash
ip addr show | grep "inet " | grep -v 127.0.0.1
```

Запам'ятайте вашу IP-адресу (наприклад, `192.168.1.100`).

### Крок 3. Відкрийте на телефоні

1. Підключіть телефон до тієї ж Wi-Fi мережі, що й комп'ютер
2. Відкрийте Chrome на телефоні
3. Введіть адресу: `http://192.168.1.100:8082`
4. Дочекайтесь завантаження додатку

### Крок 4. Встановіть як додаток (PWA)

1. В Chrome натисніть **меню** (три крапки вгорі справа)
2. Натисніть **"Додати на головний екран"** або **"Встановити додаток"**
3. Підтвердіть встановлення
4. На головному екрані телефону з'явиться іконка манула з кавою

> **Важливо:** Після встановлення додаток працюватиме офлайн (без інтернету). Інтернет потрібен тільки для AI-функцій (Gemini API).

## Варіант 2: Розміщення на хостингу (GitHub Pages, Netlify, тощо)

### GitHub Pages (безкоштовно)

1. Створіть репозиторій на GitHub
2. Завантажте всі файли проекту
3. Увімкніть GitHub Pages: Settings → Pages → Source: main, folder: / (root)
4. Додаток буде доступний за адресою: `https://ваш-логін.github.io/назва-репо/`
5. Відкрийте цю адресу в Chrome на телефоні
6. Встановіть як додаток (див. Крок 4 вище)

### Netlify (безкоштовно)

1. Зайдіть на [netlify.com](https://netlify.com)
2. Перетягніть папку `Manul` у Netlify Drop
3. Отримайте посилання і відкрийте на телефоні

---

## Варіант 3: Через USB (Chrome DevTools)

### Крок 1. Увімкніть режим розробника на телефоні

1. Налаштування → Про телефон → 7 разів натисніть "Номер збірки"
2. Налаштування → Для розробників → USB-налагодження → Увімкнути

### Крок 2. Підключіть телефон через USB

1. Підключіть USB-кабель
2. На телефоні дозвольте USB-налагодження

### Крок 3. Перенаправте порт

```bash
adb forward tcp:8082 tcp:8082
```

### Крок 4. Відкрийте на телефоні

1. В Chrome на телефоні перейдіть на `http://localhost:8082`
2. Встановіть як додаток (див. Крок 4 варіанту 1)

## Налаштування додатку

### Перший запуск

1. Відкрийте додаток
2. Введіть ваше ім'я
3. Виберіть один з варіантів:
   - **Ввести API-ключ Gemini** — для повного функціоналу з AI
   - **Спробувати демо** — працює без ключа, 3 готових уроки

### Отримання API-ключа Gemini (безкоштовно)

1. Перейдіть на [aistudio.google.com/apikey](https://aistudio.google.com/apikey)
2. Увійдіть з Google-акаунтом
3. Натисніть **"Create API Key"**
4. Скопіюйте ключ
5. Вставте його в додаток (Налаштування → API-ключ Gemini)

> **Безкоштовний план** дозволяє ~1500 запитів на день — цього вистачить для 15-20 уроків.

### Налаштування рівня підказок

В налаштуваннях (шестірня вгорі) можна змінити рівень підказок:
- **1 — Мінімум** — Манул лише каже "Майже!", без деталей
- **2 — Переклад** — показує правильну відповідь українською
- **3 — Повна допомога** — детальні пояснення граматики

### Експорт/імпорт прогресу

- **Експорт** — зберігає ваш прогрес у JSON-файл (слова, досягнення, уроки)
- **Імпорт** — відновлює прогрес з файлу (наприклад, на новому пристрої)

> API-ключ НЕ включається в експорт з безпеки.

## Усунення проблем

| Проблема | Рішення |
|----------|---------|
| "Ключ не працює" | Перевірте, що ключ скопійований повністю. Спробуйте створити новий ключ |
| Немає кнопки "Встановити" | Переконайтесь, що відкрили через HTTPS або localhost |
| Не працює озвучка | Перевірте, що в системі є німецькі голоси (Налаштування → Мови) |
| Додаток не оновлюється | Закрийте і відкрийте заново, або очистіть кеш додатку |
| Білий екран | Відкрийте DevTools (F12) → Console і перевірте помилки |
